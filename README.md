# Apex Planet Data Analytics Internship — EDA Project

## Project Overview
Exploratory Data Analysis on a simulated Indian e-commerce dataset (2023).

## Dataset
- 200 customers | 50 products | 500 orders | 1,200 order items
- Built using Python + SQLite

## Tools Used
Python, Pandas, Matplotlib, Seaborn, SQLite

## Files
- phase1_dataset_setup.py — Creates the SQLite database
- phase2_descriptive_stats.py — Summary statistics + 6 charts
- phase3_sql_business_questions.py — 7 SQL business queries + 7 charts
- phase4_multivariate_analysis.py — Correlation + advanced charts
- Ecommerce_KPI_Dashboard.pptx — Final KPI dashboard

## Key Insights
- Electronics is the top revenue category (15.9% share)
- Peak sales month: October
- Age group 36-45 has highest average order value
- Beauty products have the highest profit margin (58%)
